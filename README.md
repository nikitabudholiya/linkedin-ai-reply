## ⚡ How It Works

```
Open any LinkedIn DM
        ↓
Click ✨ Generate Reply button
        ↓
Groq AI reads the conversation
        ↓
Generates smart professional reply
        ↓
Review and hit Send!
```

## 🚀 Installation

### 1. Clone the repo
```bash
git clone https://github.com/nikitabudholiya/linkedin-ai-reply.git
```

### 2. Load in Chrome
1. Go to `chrome://extensions`
2. Enable **Developer Mode**
3. Click **Load unpacked**
4. Select the `linkedin-ai-reply` folder

### 3. Add API Key
1. Click extension icon in Chrome toolbar
2. Paste your free [Groq API key](https://console.groq.com)
3. Click **Save** ✅

### 4. Use it!
Open any LinkedIn DM and click **✨ Generate Reply**
